Este repositorio contiene mi portafolio personal, donde recopilo diferentes proyectos que voy realizando por mi cuenta para mejorar mis conocimientos en el ámbito de la informática.

El objetivo de este portafolio es mostrar de forma práctica las habilidades que voy adquiriendo durante mi formación y aprendizaje autónomo. Aquí se incluyen proyectos relacionados principalmente con áreas como:

· Cloud Computing

· Ciberseguridad

· Redes

· Sistemas

· Programación

Cada proyecto incluido en este portafolio ha sido desarrollado como parte de mi aprendizaje personal, aplicando conceptos técnicos y buenas prácticas para simular entornos reales de trabajo.

En cada carpeta o proyecto se puede encontrar documentación explicando el objetivo del proyecto, las tecnologías utilizadas y los pasos realizados durante su desarrollo.

Este portafolio irá ampliándose con nuevos proyectos a medida que continúe aprendiendo y desarrollando nuevas habilidades en el campo de la informática.
